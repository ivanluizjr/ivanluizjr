# 🍫 Brigadeiro Molds — Sea Animals

OpenSCAD scripts for 4 brigadeiro molds inspired by cute sea animals.
Design style based on **MaliLab3D** reference — animal-shaped walls like cookie cutters.

![reference](reference_style.png)

## 🦀 Animals
- 🐢 Turtle
- 🐙 Octopus
- 🐋 Whale
- 🦀 Crab

## 🔧 Mold Structure (2 pieces per animal)

| Piece | Description |
|---|---|
| **Cutter** | Tall walls (20mm) shaped like the animal — no base, open top & bottom. Press into brigadeiro to cut the shape. |
| **Press** | Same shaped walls (9mm) + solid top plate + raised details (eyes, shell, face) on bottom face. Press down to imprint design. |

## 🖨️ How to generate STL

1. Install **[OpenSCAD](https://openscad.org/)** (free)
2. Open any `.scad` file (e.g. `turtle_mold.scad`)
3. Press **`F6`** to render
4. **File → Export → Export as STL**
5. Print each piece separately

## ⚙️ Print Settings

| Setting | Value |
|---|---|
| Material | PLA |
| Layer height | 0.15 – 0.2 mm |
| Infill | 30%+ |
| Supports | Not needed |
| Color suggestion | Pastel pink `#FFB6C1` or pastel purple `#C3B1E1` |

## 📐 Dimensions

| Parameter | Value |
|---|---|
| Approximate footprint | ~60–80 mm depending on animal |
| Cutter wall height | 20 mm |
| Press wall height | 9 mm |
| Wall thickness | 1.8 mm |
| Raised detail height | 1.2 mm |

## 🍬 Usage

1. Roll brigadeiro into a ball and flatten slightly
2. Press **cutter** down to cut the animal shape
3. Press **press piece** on top to imprint details
4. Lift and plate 🎉
