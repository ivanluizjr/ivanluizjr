# 🍫 Brigadeiro Molds — Sea Animals

OpenSCAD scripts for 4 brigadeiro molds in cute sea animal designs.

## Animals
- 🐙 Octopus
- 🐋 Whale
- 🦀 Crab
- 🐢 Turtle

## Each mold has 3 parts
1. **Base mold** — open walls, flat bottom with embossed detail
2. **Detail plate** — internal embossed design plate
3. **Top press/ejector** — flat press with handle that imprints details

## How to use
1. Install [OpenSCAD](https://openscad.org/) (free)
2. Open any `.scad` file
3. Press `F6` to render
4. Export → `STL` for printing

## Print Settings
- Material: PLA
- Layer height: 0.15–0.2mm
- Infill: 30%+
- No supports needed
- Color suggestion: Pastel purple (`#C3B1E1`)

## Dimensions
- Base diameter: ~50mm
- Wall height: 18mm
- Wall thickness: 1.4mm
